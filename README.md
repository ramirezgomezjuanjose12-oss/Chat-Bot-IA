# Mago Bot - Asistente de IA para Atencion al Cliente

Este es el Mago Bot, una solucion de automatizacion real diseñada para gestionar inventarios y atencion al cliente en Mago Accesorios.

## Sobre el Desarrollador
Hola! Soy de Colombia, tengo 13 años y curso 8vo grado. Mi meta es ser Ingeniero Mecatronico. Este bot es mi primer gran proyecto uniendo Python, Inteligencia Artificial y bases de datos reales.

## Capacidades del Bot
- Consulta de Stock: Lee archivos Excel para informar disponibilidad y precios al público.
- Seguridad Garantizada: Filtra automaticamente los costos de fabrica para proteger el margen del negocio.
- Identidad Profesional: Responde con un tono serio, amable y directo.

## Tecnologias Usadas
- Python 3.12
- Gemini 2.5 Flash API
- Pandas (Manejo de datos)
- Dotenv (Seguridad de credenciales)
